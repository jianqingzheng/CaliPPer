## Notes

- **Active label**: `ACTIVITY` column (1 = active antibacterial, 0 = inactive). Defined by Wong et al. as zone-of-inhibition > 2 mm on Staphylococcus aureus.
- **Cross-dataset validation**: beta-lactam-withheld training (model never sees any beta-lactam) → test on 505 beta-lactams. This is the test that demonstrated structural-class transferability in Wong 2024.
- **Morgan FP settings**: radius 2, 2048 bits (RDKit `GetMorganFingerprintAsBitVect`); Tanimoto similarity → distance via `d = log(0.1 * (1 - sim + 0.1))`. Single chain (small molecule), no multi-chain weighting.
